DAAD v3
=======

What is DAAD V3?
----------------

DAAD V3 is the third version of the DAAD system, and the first one to be build after the 90s. It includes
new features that were not included in the original DAAD, which was DAAD V2 when it was last used by 
Aventuras AD.

New features
------------

- Flag 53 (fObjFlags) has new features in some of its bits. Not all of them are related to objects, but the
  template still refers to it as fObjFlags to avoid backward compatibility issues.

    + Bit 5: Is set if after obtaining a SL, the parser found a word that it couldn't recognize after finding the verb.
    + Bit 4: Is set if after obtaining a SL, the parser found a preposition before the first noun.
    + Bit 2: If is set, no enclicit pronouns will be applied to verbs>=240.
    + Bit 1: If is set, HASAT/HASNAT/SETAT (see below) will work on an alternative set of flags 60-91
    + Bit 0: Is set if a DOALL didn't find any objects to interate over.

Please notice is easy to change HASAT/SETAT/GETAT to work on alternative set, just by executing 
SETAT 49 1. It isn't that easy to rever to the original set tough, as you cannot access
flag 53 anymore with SETAT. So it's recommended to save the value of flag 53
somewhere else if you want to revert at anytime (i.e. COPYFF 53 200 to be able
to revert to original set with COPYFF 200 53).

- You can now use indirection in second parameter. i.e. PLACE 100 @200

- New SETAT condact. Syntax is:

  SETAT value operation

  value: points to a bit in the same range as HASAT does
  operation: says what to do with that bit: if operation is 0, the bit is cleared, if operation is 1,
             the bit is set, if operation is 2, the big is toggled.

- GETKEY is a new metacondact allowed by the compiler, it waits for a keypress and saves its value in flag
  fInkeyKey1/fInkeyKey2 (60/61). It's a metacondact because it's not a real condact, but an easier way to 
  write "PAUSE 0". In DAAD V3, when parameter for PAUSE is 0, it behaves as said above. Any other value
  behaves as usual. 


Which interpreters are compatible with DAAD V3
----------------------------------------------

- All the classic interpreters incluided in DAAD Ready have been updated to DAAD V3, that includes all ZX Spectrum targets, MSX1, CPC, C64, Plus/4, PCW, Amiga and Atari ST.
- Also, PC/DOS target using new interpreter PCDAAD supports DAAD V3.
- Also, HTML target using new interpreter PCDAAD supports DAAD V3.
